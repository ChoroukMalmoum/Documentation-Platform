Augmenter

class Augmenter(image, label, augmentation_ids, rotaion_degrees=180, crop_size=50, shear=(-15, 15, -15, 15), shear_degress=1, hue=(-0.1, 0.1), saturation=(0, 3), brightness=(0, 3), gaussian_blur_kernel_size=15)

Class for data augmentation on images and bounding boxes.

Parameters:

  • image (BytesIO) – BytesIO object containing image data.

  • label (BytesIO) – BytesIO object containing label data.

  • augmentation_ids (list) – List of indices representing augmentations to apply.

  • rotaion_degrees (int, optional) – Rotation degrees for RandomRotation. Defaults to 180.

  • crop_size (int, optional) – Size of the crop for RandomCrop. Defaults to 50.

  • shear (tuple, optional) – Shear range for RandomAffine. Defaults to (-15, 15, -15, 15).

  • shear_degress (int, optional) – Degrees for shear in RandomAffine. Defaults to 1.

  • hue (tuple, optional) – Range of hue adjustment for ColorJitter. Defaults to (-0.1, 0.1).

  • saturation (tuple, optional) – Range of saturation adjustment for ColorJitter. Defaults to (0, 3).

  • brightness (tuple, optional) – Range of brightness adjustment for ColorJitter. Defaults to (0, 3).

  • gaussian_blur_kernel_size (int, optional) – Kernel size for GaussianBlur. Defaults to 15.

Note

The torchvision.transforms module is used for image transformations. For more information, refer to the torchvision documentation.

Note

The cv2 (OpenCV) library is used for image processing. For more information, refer to the OpenCV documentation.

Note

The torchvision.tv_tensors.BoundingBoxes class is used for representing bounding boxes. For more information, refer to the torchvision documentation.
