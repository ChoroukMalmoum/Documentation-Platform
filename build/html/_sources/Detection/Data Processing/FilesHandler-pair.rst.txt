FilesHandler.pair
++++++++++++++++++

.. method:: FilesHandler.pair(images, labels)

   Pairs image and label files based on their names.

   :param images: List of image file-like objects.
   :type images: list
   :param labels: List of label file-like objects.
   :type labels: list

   :return: A list of tuples, each containing a paired image and label file.
   :rtype: list
