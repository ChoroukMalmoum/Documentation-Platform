CustomYoloModel.add_conv_layer_before_maxpool

CustomYoloModel.add_conv_layer_before_maxpool(kernel_size, out_channels, stride, target_maxpool, padding=None)

Adds a convolutional layer before a MaxPool layer in the architecture.

Parameters:

  • kernel_size (int) – The size of the kernel. Must be an odd number.

  • out_channels (int) – The number of output channels.

  • stride (int) – The stride of the convolution.

  • target_maxpool (int) – The index of the MaxPool layer before which to add the convolutional layer.

  • padding (int, optional) – The padding of the convolution. Default is (kernel_size-1) // 2.

Raises:

IndexError – If the index is out of range.
