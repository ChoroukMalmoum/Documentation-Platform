Merge Datasets

merge_datasets(dataset1, dataset2, column_dataset1, column_dataset2, merge_type='left')

The merge_datasets() method merges two datasets based on specified columns.

Parameters:

  • dataset1 (pd.DataFrame) – The first dataset.

  • dataset2 (pd.DataFrame) – The second dataset.

  • column_dataset1 (str) – The column from the first dataset for merging.

  • column_dataset2 (str) – The column from the second dataset for merging.

  • merge_type (str, optional) – Type of merge to be performed. Options: ‘left’, ‘right’, ‘inner’, ‘outer’.

Returns:

Merged dataset.

Return type:

pd.DataFrame

  • Refer to the pandas.DataFrame.merge documentation for more details on the merge method.
