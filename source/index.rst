Overview

Welcome to the documentation for Our Application ! This document provides an introduction and overview of the features and functionality offered by our application.

Tabular data
---------------
.. _tabular_data: 

The `Tabular data <Tabular_data/overview.html>`_ functionality is designed to assist users in working with tabular data. It offers a seamless workflow for cleaning, transforming, and exploring data, as well as creating insightful visualizations and making predictions using various machine learning techniques.

Classification
---------------
.. _classification: 

The `Classification <Classification/overview.html>`_ functionality is designed to streamline image classification tasks by providing users with a versatile and user-friendly environment. As a key component within the GIIADS platform, our Image Classification application simplifies the complexities associated with image classification tasks, offering users a flexible and empowering experience.


Detection
---------------
.. _detection:

The `Detection <Detection/overview.html>`_ functionality is designed to assist users in object detection tasks. Our platform enables users to effortlessly train custom object detection models by uploading labeled images. With a user-friendly interface and a choice of pre-existing models, users can quickly and effectively train models for accurate predictions. Once trained, the models can be applied to new images, making object detection tasks accessible to users of all technical backgrounds.
