Yolo.train.train_Custom_model

Yolo.train.train_Custom_model(model, learning_rate=2e-5, batch_size=8, epochs=10, weight_decay=0.0005, optimizer='Adam', S=7, B=2, C=2)

Trains a custom YOLO model.

This function trains a custom YOLO model on a dataset. The dataset is loaded from a CSV file and transformed into the appropriate format. The function also calculates the mean average precision (mAP) of the model after each epoch and saves the model if the mAP is greater than 0.9.

Parameters:

  • model (nn.Module) – The model to train.

  • learning_rate (float, optional) – The learning rate for the optimizer. Default is 2e-5.

  • batch_size (int, optional) – The batch size for the DataLoader. Default is 8.

  • epochs (int, optional) – The number of epochs to train for. Default is 10.

  • weight_decay (float, optional) – The weight decay for the optimizer. Default is 0.0005.

  • optimizer (str, optional) – The optimizer to use. Can be either ‘Adam’ or ‘SGD’. Default is ‘Adam’.

  • S (int, optional) – The number of cells in the output grid. Default is 7.

  • B (int, optional) – The number of bounding boxes per cell. Default is 2.

  • C (int, optional) – The number of classes. Default is 2.

Raises:

ValueError – If the optimizer is not ‘Adam’ or ‘SGD’.
